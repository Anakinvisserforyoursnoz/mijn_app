# Modern PyQt6 Calculator

Modern Python applications developed on Linux Mint by a Dutch Linux enthusiast and open-source developer 🇳🇱🐧

## About

A modern calculator application built with Python and PyQt6 on Linux Mint.

This repository also includes a real-time Discord-like chat application built with Python sockets and Tkinter.

---

## Features

### Calculator App

- Modern dark mode UI
- PyQt6 interface
- Basic calculations
- Stylish buttons
- Responsive layout

### Discord-like Chat App

- Real-time messaging
- Multi-user support
- TCP socket networking
- Discord-style dark theme
- JSON messaging system
- Threaded networking
- Multiple channels UI

## Installation

Clone the repository:

```bash
git clone https://github.com/Anakinvisserforyoursnoz/mijn_app.git
cd mijn_app
```

Create a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install PyQt6
```

---

## Run the Calculator

```bash
python3 calculator.py
```

---

## Run the Chat Server

```bash
python3 server.py
```

---

## Run the Chat Client

Open another terminal:

```bash
python3 app.py
```

Use:

- Username
- IP address
- Port `5555`

For local testing:

```text
127.0.0.1
```
