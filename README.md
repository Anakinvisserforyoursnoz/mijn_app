## Installatie

Clone de repository:

```bash
git clone https://github.com/Anakinvisserforyoursnoz/mijn_app.git
cd mijn_app
```

Maak virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

Installeer dependencies:

```bash
pip install PyQt6
```

